Projeto - Interfaces de autenticação - App Leankeep

Detalhes do projeto

  • Categoria: UX, UI e Desenvolvimento Front-end
  • Empresa: Leankeep Facility Software
  • Data do projeto: 12/2020
  • Website: leankeep.com.br

Criação de novas interfaces de autenticação, login, cadastro e recuperação de senha. Nesse projeto é perceptivel a mudança e como foi possível deixar a interface muito mais clean, trazendo uma melhor fluidez dos elementos. Ainda algo que destaco é a contante comunicação com o usuário durante a sincronização de dados, isso faz com que o tempo se torne "mais curto" do ponto de vista do usuário tendo em vbista que ele não irá ficar estagnado olhando para uma tela sem saber o que acontece e sim irá acompanhar o que está ocorrendo por traz da sincronização.
